# BPSD 30 天升级先验 — 文献依据摘要

## 假设 H

未来 30 天内 BPSD 临床显著恶化：NPI 总分较基线升高 ≥4 分，或任一 NPI 分项乘积 ≥4（与 JAMA CHS 及多项试验入组标准一致）。

## 推导公式

P(H|stage) ≈ p_stage × λ_stage

- p_stage：该期 BPSD/NPI 患病率（文献）
- λ_stage：月尺度症状新发或加重比例（纵向研究启发式）

## 各阶段先验

| 阶段 | P(H) |
|------|------|
| 1 早期 | 6% |
| 2 轻度 | 14% |
| 3 中度 | 26% |
| 4 中重度 | 36% |
| 5 重度 | 50% |
