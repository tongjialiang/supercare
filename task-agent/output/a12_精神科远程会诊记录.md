**精神科远程会诊记录**

**会诊日期**：  

**参会人员**：  

**问题摘要**：  
陈女士老人返院一周内整体状态平稳，BPSD症状持续缓解（NPI评分15分），仅发生1次轻微行为事件且干预有效。然而，近2天（返院第6、7天）照护员观察到老人在自主练习跳舞时出现短暂记忆空白，表现为突然忘记舞步、停顿发呆，持续约1–2分钟后自行恢复，恢复后无躯体不适但情绪略显低落，并开始拒绝与其他老人共同跳舞及交流。家属探视时发现该现象，高度担忧其认知功能加速下降，并忧虑此症状可能诱发BPSD复发，导致原本缓解的焦虑情绪再度加重，多次向照护团队咨询认知状态评估及干预措施。

**会诊结论**：  
结合老人既往“阿尔茨海默病性痴呆6年，存在近期记忆减退、时间/地点定向力下降”的诊断基础，当前新发的短暂记忆空白、任务中断及社交退缩，符合痴呆进程中常见的认知波动表现，属于疾病自然进展的一部分。此类波动可能成为BPSD复发的前驱信号，尤其当伴随情绪低落与社会参与减少时，需高度警惕抑郁情绪或淡漠综合征的发展。目前尚无急性精神病性症状或激越行为，BPSD总体处于可控状态，但存在复发风险。

**药物调整建议**：  
1. 维持现有精神科药物方案不变：继续规律服用多奈哌齐、美金刚以稳定认知功能；奥氮平片1.0mg qn及劳拉西泮2.0mg qn用于BPSD预防性管理，当前剂量合理且有效，暂无需调整。  
2. 建议药师核查近一周实际服药依从性，排除因漏服或重复记录导致的认知药物暴露不足（参考历史用药洞察提示可能存在记录重复问题）。  
3. 暂不新增抗抑郁或镇静药物，避免过度用药加重认知抑制或跌倒风险。

**非药物干预建议**：  
1. **优化舞蹈活动形式**：将独自练习调整为结构化小组活动，由照护员陪伴引导，使用视觉提示卡（如舞步图示）辅助记忆，降低任务失败带来的挫败感。  
2. **引入音乐与怀旧疗法**：结合老人喜爱的音乐开展回忆对话，增强情感联结与自我价值感，缓解情绪低落。  
3. **建立规律日程表**：固定每日活动安排，减少环境不确定性，提升安全感与定向能力。  
4. **情绪支持干预**：照护员在老人出现发呆后主动温和陪伴，避免追问或催促，采用共情式语言（如“刚才有点累了吧？我们休息一下”）降低其羞耻感。  
5. **家属协同参与**：鼓励家属探视时参与轻柔互动（如一起听歌、简单手势舞），避免聚焦于“记不记得”等引发焦虑的问题。

**后续观察重点**：  
- 记忆空白发作频率、持续时间及是否扩展至其他日常活动（如用餐、洗漱）；  
- 情绪状态变化（是否出现持续悲伤、哭泣、言语消极）；  
- 社交意愿与参与度（是否进一步退缩至房间、拒绝基本交流）；  
- BPSD症状是否复发（如激越、攻击、幻觉、游走等）；  
- 日常生活能力是否出现新发依赖。

**下次复诊时间**：  
建议1周后（返院第14天左右）进行精神科远程复诊，评估干预效果及认知情绪状态。

**待执行事项**：  
1. 照护团队详细记录每次记忆空白事件的时间、情境、持续时间及后续行为反应；  
2. 安排一次家属沟通会，由精神科医师或个案管理师解释当前认知波动的临床意义，缓解其对“快速恶化”的误解，提供家庭支持策略；  
3. 康复治疗师于3日内调整舞蹈干预方案，提交修订版活动计划；  
4. 药房核查近7天实际发药与服药记录，确认认知药物依从性。

**备注**：  
老人既往存在皮肤完整性受损、心动过缓（心率常处正常下限）、潜在凝血风险等健康隐患（参考历史风险洞察），虽非本次会诊核心问题，但提醒多学科团队在实施新干预时注意安全防护，避免因活动调整增加跌倒或碰撞风险。当前生命体征平稳、血压血脂控制良好，为非药物干预提供了良好生理基础。
