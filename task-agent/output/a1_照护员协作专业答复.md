# 照护协作建议报告：陈女士（上海籍失智老人）

## 一、问题归纳
1. **行为抗拒**：老人存在阿尔茨海默病相关BPSD症状，明确拒绝传统康复训练，影响功能维持。
2. **高跌倒风险**：近期发生右额部皮下血肿，合并体位性低血压、行动能力下降及夜间频繁离床（近3日离床4–9次/晚），浴室洗澡需协助，环境安全防护不足。
3. **用药管理隐患**：联用多奈哌齐、美金刚、奥氮平、劳拉西泮等多种精神科与慢病药物，存在重复录入、名称错误及潜在相互作用风险。
4. **家属焦虑突出**：家属照护意愿强但长期处于焦虑状态，需增强信任与参与感。

## 二、处理步骤（可执行建议）

### （1）基于喜好的非药物干预方案（替代康复训练）
- **将“跳舞”转化为结构化活动**：
  - 每日上午康乐活动时段（原安排基础上）增设15–20分钟“怀旧舞蹈时光”，播放老人熟悉的沪剧、老歌伴奏，鼓励站立轻舞或坐姿手部律动；
  - 若老人抗拒集体活动，可改为一对一陪伴式舞蹈互动，由照护员牵手引导简单步伐，强调“一起玩”而非“训练”；
  - 下午可安排音乐聆听+节奏拍打（如沙锤、鼓），维持运动协调性。
- **避免强制参与**：若老人当日情绪不佳，不强行推进，转为散步或园艺等低刺激活动，保持正向体验。

### （2）24小时防跌倒闭环管理
- **夜间重点防护**：
  - 实施“夜间如厕全程陪护”：老人有夜间离床习惯（近3日平均离床6次以上），需照护员在床边值守或使用离床感应垫联动呼叫；
  - 床旁至卫生间路径确保无障碍、防滑、照明充足（建议安装感应夜灯）；
  - 卫生间加装L型扶手、防滑垫，淋浴椅固定稳妥。
- **日间防护**：
  - 起身遵循“坐30秒→站30秒→行走”原则，监测体位性低血压；
  - 洗澡时必须两人协助（一人递物、一人保护平衡），水温控制在38–40℃，避免烫伤。

### （3）用药规范落实
- **立即启动药师审查**：
  - 整理当前用药清单（含多奈哌齐、美金刚、奥氮平、劳拉西泮、振源片等），提交药师核查重复项（如“多琋磷脂酰胆碱胶囊”疑似重复录入）、剂量合理性及相互作用；
  - 统一药品名称，制作图文版服药卡（含药片照片、服用时间、目的）。
- **服药执行**：
  - 每次服药由两名照护员双人核对，记录实际服用情况；
  - 观察服药后反应（如嗜睡、步态不稳），尤其晚间服用劳拉西泮后加强看护。

## 三、风险防护要点
| 风险类型 | 防护措施 |
|--------|--------|
| **跌倒/外伤** | 夜间全程陪护如厕；浴室防滑改造；避免单独行走；起搏器区域避免碰撞 |
| **药物不良反应** | 清理重复用药；监测白细胞/血小板（历史提示偏低）；观察锥体外系反应（奥氮平） |
| **BPSD激惹** | 不否定、不争辩；用兴趣转移注意力；保持环境安静、光线柔和 |
| **睡眠紊乱** | 固定作息，避免午后长时间卧床；晚间减少液体摄入；非必要不使用安眠药 |

## 四、家属沟通策略
- **主动沟通频率**：每周至少1次结构化沟通（电话或面谈），通报干预进展。
- **沟通内容建议**：
  - “我们发现陈阿姨特别喜欢跳舞，现在把康复动作融入她熟悉的旋律中，今天她跟着《天涯歌女》摆手笑了，这是很好的信号。”
  - “为了预防再次跌倒，我们加强了夜间陪护，也请您放心，她的起搏器区域我们会特别注意保护。”
  - “用药清单正在请药师审核，下周会和您确认最终方案，确保安全有效。”
- **邀请参与**：鼓励家属提供老人年轻时喜爱的歌曲清单，或录制家庭舞蹈视频用于怀旧疗法，增强情感联结。

## 五、反馈要点（照护员每日记录）
1. **舞蹈/音乐活动参与情况**：是否参与、持续时间、情绪反应（愉悦/烦躁/拒绝）；
2. **跌倒相关事件**：离床次数、是否陪护、有无失衡或跌倒；
3. **用药执行**：是否按时服药、有无漏服/拒服、服药后异常表现；
4. **家属互动反馈**：家属提出的问题、情绪变化、合作意愿；
5. **睡眠质量**：实际睡眠时长、夜间觉醒原因（如如厕、找人、不安）。
