## 五、后续观察重点  
1. **BPSD行为演变**：密切监测是否出现言语攻击、肢体抗拒、游走、昼夜颠倒等升级行为。  
2. **康复依从性**：记录每日对认知/运动干预的配合程度，识别拒绝诱因（如疲劳、环境刺激、躯体不适）。  
3. **家属情绪支持**：主动沟通老人病情进展，提供BPSD应对指导，缓解其焦虑，避免家属过度干预影响照护一致性。  
4. **基础健康风险**：  
   - 心率及循环状态（尤其日间活动时）；  
   - 皮肤完整性（预防擦伤/淤青）；  
   - 胃肠道功能（警惕呕吐、腹泻复发）。  
5. **非药物干预优化**：结合老人喜好（如跳舞），设计更柔性、短时、高愉悦感的活动，提升参与意愿。
